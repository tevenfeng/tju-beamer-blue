tju-beamer-blue
==============
Beamer LaTex templete for Tianjin University (blue themed) 天津大学Beamer LaTex模板

使用需求
--------------
仅在Windows下安装了TexLive的环境中测试通过

使用说明
--------------
### 编译
可使用其中的 make.bat（默认使用xelatex进行编译）

也可使用pdflatex进行编译

### 支持功能
* 中文不乱码的书签
* Beamer 支持的应该都支持...
如 block, list, numbered list, pause
